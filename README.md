# Website Status Checker with HTTP Error Handling

A robust Python tool for checking the availability and response time of websites with comprehensive HTTP error handling.

## Features

- Check multiple websites concurrently
- Comprehensive HTTP error handling (timeouts, connection errors, SSL errors, etc.)
- Export results to JSON and CSV formats
- Configurable timeout and concurrent workers
- Detailed logging
- Color-coded console output
- Statistics summary
- SSL verification control
- Custom user agent support
